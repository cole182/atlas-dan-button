# Atlas UI Design Ideas

## Chosen Approach: Dark Intelligence Terminal

**Design Movement**: Data-forward dark ops / Bloomberg Terminal meets modern SaaS

**Core Principles**:
1. Dark charcoal/near-black base with amber/orange accent (matching existing Atlas branding)
2. Data density — every pixel earns its place, no wasted whitespace
3. Sidebar navigation with module cards, each feeling like a live mission briefing
4. Subtle red/amber glow effects on active states to signal urgency

**Color Philosophy**:
- Background: `#0D0B0A` (near black, warm undertone)
- Surface: `#1A1614` (card backgrounds)
- Border: `#2E2420` (subtle warm borders)
- Accent Primary: `#E8521A` (EBRE orange-red)
- Accent Secondary: `#F5A623` (amber for data highlights)
- Text Primary: `#F5F0EB` (warm white)
- Text Muted: `#8A7A72` (warm gray)
- Success: `#22C55E`
- Warning: `#F59E0B`
- Danger: `#EF4444`

**Typography**:
- Display: `Syne` (bold, geometric — for module titles)
- Body: `Inter` (clean, readable)
- Mono: `JetBrains Mono` (for data/stats)

**Layout**:
- Fixed left sidebar (64px collapsed, 240px expanded)
- Main content area with module-specific layouts
- Top bar with user info, notifications, live status

**6 Modules**:
1. County Scraper (Atlas Core) — daily lead feed
2. Property Condition Scoring — CSV upload + AI scoring
3. Insurance Gap Finder — API-powered risk analysis
4. Social Distress Signal — Reddit/FB/NextDoor monitor
5. Obituary Monitor — probate cross-reference
6. Fire Damage Monitor — fire report tracker
